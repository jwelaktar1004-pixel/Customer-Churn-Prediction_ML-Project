# Customer Churn Prediction using Machine Learning

## Overview
This project focuses on predicting customer churn in the telecom industry using machine learning techniques.  
The goal is to identify customers likely to leave the service so businesses can take proactive retention actions.

## Key Steps
- Performed exploratory data analysis to identify churn patterns  
- Preprocessed data and encoded categorical features  
- Handled class imbalance using SMOTE  
- Trained and evaluated multiple ML models  

## Models Used
- Decision Tree  
- Random Forest  
- XGBoost  

## Outcome
The final model helps identify high-risk customers and supports data-driven retention strategies.
